\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

In the context of the rapid growth of online shopping---especially among busy young parents
who value convenience and trustworthy product information---the need for well-designed
e-commerce platforms in the children's fashion segment has become increasingly important.
Motivated by this trend, this project aims to develop an online retail website for kids'
clothing and accessories, \brandName{}, that meets both customer shopping needs and seller
management requirements.

The system is built on \textbf{Next.js 14} (App Router) as a full-stack application using React
Server Components and Server Actions, with \textbf{Prisma ORM} connecting to a
\textbf{PostgreSQL} database hosted on \textbf{Supabase}. The website provides the core features
of an e-commerce platform: product display and management with variants (size, color), search
and filtering, shopping cart, order placement, and online payment through the \textbf{PayOS}
gateway alongside cash on delivery (COD), together with a full admin panel for managing
products, inventory, orders, promotions, and revenue reports.

Beyond the basic features, the system also implements advanced functionalities such as moderated
product reviews and ratings, wishlist and product comparison, discount vouchers and flash sales,
return/refund management, automated transactional emails, and optimizations for SEO, performance,
and security. The interface is designed to be user-friendly and responsive across devices.

Throughout the development process, I applied and strengthened my knowledge of system analysis
and design, modern web programming, database design, application security, and software
deployment, as well as essential skills for managing and delivering a complete software project.

\vspace{8pt}
\noindent\textbf{Keywords:} \brandName, e-commerce, Next.js, Prisma, PostgreSQL, Supabase, PayOS.
